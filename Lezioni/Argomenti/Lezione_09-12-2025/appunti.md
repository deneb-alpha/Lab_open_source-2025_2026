	## APPUNTI GITHUB -
	
	git fetch --all= Va a risincronizzare tutte le nuove informazioni create Esempio:(branch).
	
	git checkout -b ///= possiamo utilizzare il fork e crearci una branch.

	git branch= per vedere la branch.

	git branch -vv= vedo le informazioni della branch Esempip:(dove mi trovo).

	git branch -a= in verde trovo le branch dove sono ora, in rosso i remotes.

	git commit= -m(messaggio), per creare un commit.

	git push= inserisce la branch nelle etichette.

	git pull= aggiorna in locale quello che abbiamo su remoto.

	git diff= mi fa vedere le modifiche che ho apportato al commit.

	git status= vedo lo stato della branch.

	git add adh.md= creo un file di testo.

	git log = dopo aver creato commit per vedere il commit.
